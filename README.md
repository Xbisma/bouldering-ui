# Bouldering UI Project

## Description  
A simple bouldering-themed UI project built to practice layout, styling, and responsiveness. Focuses on clean design and making the interface work well on different screen sizes.

## Features  
- Responsive layout  
- Clean and minimal design  
- Basic UI structure for a bouldering theme  

## Tech Used  
- HTML  
- CSS  

## Purpose  
This project was built for practice to improve front-end skills and understanding of responsive design.

## How to Run  
1. Clone the repository  
2. Open the `index.html` file in your browser
